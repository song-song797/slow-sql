### 检索条件组件设计

#### 高级检索面板
- **关键词搜索**: 支持 SQL 语句全文搜索
- **数据库筛选**: 多选下拉框
- **表名筛选**: 自动完成输入框
- **执行时间范围**: 数字输入框（最小值-最大值）
- **时间范围**: 日期时间选择器
- **状态筛选**: 单选下拉（全部/待分析/分析中/已完成/失败）
- **应用名称**: 文本输入框
- **风险等级**: 多选下拉（低/中/高/严重）

#### 快速筛选标签
- 今日慢 SQL
- 超过 5 秒的 SQL
- 未分析的记录
- 高风险 SQL

### 结果列表组件设计

#### 表格列
1. **复选框**: 批量操作
2. **ID**: 记录 ID（可点击查看详情）
3. **SQL 预览**: 截断显示前 100 字符，悬停显示完整 SQL
4. **数据库**: 数据库名称
5. **表名**: 涉及的表（标签形式）
6. **执行时间**: 带颜色标识（绿色<1s, 黄色1-3s, 红色>3s）
7. **记录时间**: 格式化时间显示
8. **状态**: 状态标签（带颜色）
9. **操作**: 
   - 查看详情
   - 执行分析
   - 下载报告（如果已分析）

#### 批量操作
- 批量分析
- 批量导出
- 批量删除

## 5. 组件设计规范

### 颜色方案
- **成功/正常**: 绿色 (#52c41a)
- **警告**: 橙色 (#faad14)
- **错误/高风险**: 红色 (#ff4d4f)
- **信息**: 蓝色 (#1890ff)
- **待处理**: 灰色 (#d9d9d9)

### 状态标签样式
- **待分析**: 灰色圆角标签
- **分析中**: 蓝色圆角标签 + 加载动画
- **已完成**: 绿色圆角标签
- **失败**: 红色圆角标签

### 风险等级样式
- **低**: 绿色标签
- **中**: 橙色标签
- **高**: 红色标签
- **严重**: 深红色标签 + 警告图标

### 交互设计
- **悬停效果**: 所有可点击元素添加悬停高亮
- **加载状态**: 异步操作显示加载动画
- **空状态**: 无数据时显示友好的空状态提示
- **错误处理**: 错误信息以 Toast 或 Modal 形式展示

## 6. 响应式设计

### 断点设置
- **移动端**: < 768px
- **平板**: 768px - 1024px
- **桌面**: > 1024px

### 移动端适配
- 检索条件面板改为抽屉式
- 表格改为卡片式布局
- 操作按钮改为图标按钮
