# 慢 SQL 分析系统 - 设计概览

## 项目概述

慢 SQL 分析系统是一个用于记录、检索、分析和优化慢 SQL 查询的 Web 应用系统。系统提供强大的检索功能、智能分析和详细的优化建议。

## 核心功能

### 1. 慢 SQL 记录管理
- ✅ 完整的 SQL 信息记录
- ✅ 执行计划和性能指标
- ✅ 数据库上下文信息
- ✅ 执行环境信息

### 2. 强大的检索功能
- ✅ 多条件组合检索
- ✅ 关键词全文搜索
- ✅ 时间范围筛选
- ✅ 状态和风险等级筛选
- ✅ 分页和排序

### 3. 智能分析
- ✅ 自动分析慢 SQL 根本原因
- ✅ 生成优化建议
- ✅ 风险等级评估
- ✅ PDF 报告生成

### 4. 结果管理
- ✅ 分析结果查看
- ✅ 优化建议展示
- ✅ PDF 报告下载
- ✅ 结果检索和筛选

## 技术架构

### 前端技术栈
- **框架**: React + TypeScript
- **UI 组件库**: Ant Design (推荐)
- **状态管理**: Redux / Zustand (可选)
- **路由**: React Router
- **HTTP 客户端**: Axios
