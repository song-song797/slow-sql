## 4. 数据流转

### 4.1 记录创建
```
外部系统/日志 → 数据采集 → 数据清洗 → 入库存储
```

### 4.2 分析任务
```
用户触发 → 任务创建 → 任务队列 → 分析引擎 → 结果存储 → 通知用户
```

### 4.3 报告生成
```
分析结果 → 模板渲染 → PDF 生成 → 文件存储 → 下载链接
```

## 5. 状态转换

### 记录状态
```
pending → analyzing → completed
                ↓
             failed
```

### 分析任务状态
```
pending → processing → completed
              ↓
           failed
```

## 6. 用户操作流程

### 6.1 查看慢 SQL 记录
1. 进入检索页面
2. 设置检索条件（可选）
3. 点击搜索
4. 查看结果列表
5. 点击记录查看详情

### 6.2 执行分析
1. 选择记录（单个或批量）
2. 点击"执行分析"按钮
3. 等待分析完成（可查看进度）
4. 查看分析结果
5. 下载 PDF 报告

### 6.3 查看分析结果
1. 进入分析结果页面
2. 查看分析概览
3. 阅读根本原因分析
4. 查看优化建议
5. 下载 PDF 报告
